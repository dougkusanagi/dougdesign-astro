# 03. Fluxo Editorial

1. Ler o inventario e rodar `dougseo intent check`.
2. Criar ou atualizar o arquivo em `src/content/blog/`.
3. Preencher frontmatter editorial.
4. Inserir links internos contextuais no corpo.
5. Gerar a capa.
6. Rodar `dougseo audit`.
7. Publicar agora com `dougseo publish` ou agendar com `dougseo schedule`.
8. Commitar e dar push para disparar o deploy.
