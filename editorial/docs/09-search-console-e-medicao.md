# 09. Search Console e Medicao

- Use `dougseo search-console inspect --latest 20` para revisar indexacao recente.
- Use `dougseo search-console performance --days 28` para ler clicks, queries, pages e tendencia recente.
- Prefira `GOOGLE_SEARCH_CONSOLE_SERVICE_ACCOUNT_JSON_B64` em `.env`; o CLI renova o access token automaticamente.
- Priorize correcoes de CTR, canibalizacao e gaps de cluster.
- Ao fim de uma rodada, registre quais URLs foram publicadas ou agendadas.
