# 04. Frontmatter e Templates

Campos obrigatorios para novos posts:

- `title`
- `slug`
- `pubDate`
- `author`
- `category`
- `draft`
- `scheduled`
- `meta_description`
- `description`
- `image`
- `featured_image`
- `keyword_principal`
- `content_type`
- `cluster`
- `assunto`
- `intencao_busca`
- `decisao_do_leitor`
- `fato_novo`
- `canonical_role`
- `internal_links`
- `canibalizacao`
- `fontes_oficiais`

Use `dougseo post scaffold` para gerar a estrutura inicial correta.
