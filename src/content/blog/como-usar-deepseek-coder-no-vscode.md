# Como Usar o DeepSeek Coder no VSCode para Programacao

## Resumo rapido

Para usar o **DeepSeek Coder** no VSCode, instale uma extensão de cliente de IA compatível, como a *Continue* ou a *Double.bot*, insira a sua chave de API oficial do DeepSeek ou conecte-se a uma instância local via Ollama. Essa integração permite autocompletar código em tempo real, gerar refatorações e tirar dúvidas direto no editor de código com alta velocidade e baixo custo.

---

## O que e o DeepSeek Coder?

O DeepSeek Coder é um modelo de Inteligência Artificial altamente especializado em programação. Ele rivaliza diretamente com os modelos mais famosos do mercado, oferecendo excelentes resultados em linguagens populares como Python, JavaScript, Java e C++.

Diferente de modelos mais gerais, o DeepSeek foi treinado especificamente em repositórios de código aberto e documentações técnicas, o que o torna ideal para acompanhar ferramentas de desenvolvimento modernas. Essa precisão é ideal quando você está configurando novos ambientes de trabalho ou escrevendo código estruturado, como nos builds de alta performance do [Astro 7 para desenvolvimento web](https://dougdesign.com.br/novidades-astro-7-desenvolvimento-web/).

## Passo a passo para configurar no VSCode

A maneira mais eficiente e estável de integrar o modelo hoje é utilizando a extensão **Continue**:

1. **Instale a Extensão:** No VSCode, abra o painel de extensões (Ctrl+Shift+X) e pesquise por **Continue**. Instale o plugin oficial.
2. **Obtenha a Chave de API:** Acesse o painel oficial da DeepSeek e crie uma nova chave de API (API Key).
3. **Configure o Arquivo `config.json`:** No painel da extensão Continue, selecione a engrenagem de configurações. Adicione o bloco correspondente ao DeepSeek Coder informando a sua API key:
   ```json
   {
     "model": "deepseek-coder",
     "provider": "deepseek",
     "apiKey": "SUA_API_KEY_AQUI"
   }
   ```
4. **Pronto para Usar:** Use o atalho `Ctrl + I` para abrir o chat de edição na linha atual ou `Ctrl + L` para adicionar trechos de código selecionados ao painel lateral do chat explicativo.

Ao trabalhar com IA no terminal ou no editor, a segurança do código fonte continua sendo prioridade. Certifique-se de compreender os perigos de dados maliciosos ao ler sobre [como se proteger de prompt injection em IA](https://dougdesign.com.br/como-se-proteger-de-prompt-injection-ia/).
