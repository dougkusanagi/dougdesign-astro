## Microsoft 365 Copilot recebe atualização para facilitar o uso de prompts

O Microsoft 365 Copilot, assistente inteligente da Microsoft para aplicativos de escritório, está ainda mais fácil de usar. A novidade, anunciada pela empresa nesta terça-feira (29), traz sugestões de prompts para o chatbot, eliminando a necessidade de o usuário pensar em comandos complexos para aproveitar ao máximo a inteligência artificial.

Agora, ao ativar o Copilot em um documento, planilha ou apresentação, o usuário será recebido com exemplos de prompts que podem ser usados para iniciar a interação com a IA. As sugestões serão personalizadas para o tipo de conteúdo que está sendo visualizado, adaptando-se ao contexto do trabalho.

## Sugestões personalizadas para um fluxo de trabalho mais intuitivo

Por exemplo, se o usuário estiver trabalhando em uma apresentação sobre um novo produto, o Copilot poderá sugerir prompts como "Crie uma introdução cativante" ou "Gere um resumo executivo conciso". A ideia é fornecer um ponto de partida para o usuário, que poderá editar o prompt sugerido ou criar um novo do zero.

## Além dos prompts: outras melhorias no Copilot

Além das sugestões de prompts, a Microsoft também está implementando outras melhorias no Copilot, incluindo:

### Melhorias na geração de texto

O Copilot está sendo atualizado para gerar textos mais criativos, informativos e envolventes.

### Suporte a novos idiomas

A Microsoft está expandindo o suporte a idiomas para o Copilot, tornando-o acessível a um público ainda maior.

### Integração aprimorada com aplicativos do Microsoft 365

O Copilot está sendo integrado de forma mais profunda aos aplicativos do Microsoft 365, como Word, Excel, PowerPoint e Outlook, para fornecer uma experiência mais coesa e intuitiva.

## Disponibilidade

As novas funcionalidades do Copilot estão sendo liberadas gradualmente para os usuários do Microsoft 365. A expectativa é que a atualização esteja disponível para todos os usuários nas próximas semanas.

## Revolucionando a maneira como trabalhamos

Com essas atualizações, a Microsoft busca tornar o Copilot ainda mais acessível e útil para os usuários, independentemente de sua familiaridade com prompts e inteligência artificial. A empresa acredita que o Copilot tem o potencial de revolucionar a maneira como as pessoas trabalham, tornando-as mais produtivas, criativas e eficientes.
